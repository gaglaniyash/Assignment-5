HTML (Hyper Text Markup Language)

HTML is the standard markup language for creating web pages. It describe the structure of s web page using markup.

  • Block Elements: h1, p, ul, col
  • Inline Elements: i, strong, img
  • Attributes: type, src, alt, href
Tag Description
h1 Main Heading
p Paragraph
ul Unordered List

CSS (Cascading Style Sheets)

CSS is used for styling web pages. It describes how elements should be displayed on the screen.

  • Font Properties: font-family, font-size, color
  • Box Model: margin, padding, border
  • Text Properties: text-align, line-height, text-decoration
Property Description
font-family Defines the font of the text
color Sets the text color

JavaScript

Javscript is a programming language that enables interactive web pages. It can dynamically modify content.

  • Variables and Data Types
  • Functions and Control Structures
  • DOM Manipulation
Concept Description
Variables Containers for storing data values
color Resuable blocks of code
